Course Management System by Kaustubh Kulkarni-----------------

A Java Web Application built using JSP, Servlets, and MySQL.

Overview------------

The Course Management System (CMS) is a simple web-based application that allows administrators and students to manage courses, enrollments, and user data.
Built with Java, JSP, Servlets, and MySQL, the system follows an MVC-like structure and uses basic session management for authentication.

Features--------------------
1.  Admin
Add, update, and delete courses
Manage student accounts
View total enrollments

2. Students
Register & Log in
View available courses
Enroll in courses
View personal dashboard

3. Technologies Used
Java (JSP & Servlets)
HTML, CSS, Bootstrap
MySQL Database
JDBC
Apache Tomcat 9.0
